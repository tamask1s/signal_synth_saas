# Synsigra Private Beta Support and Service Expectations

Effective date: 2026-07-11

## Support channel

Open a support request at:
https://github.com/tamask1s/signal_synth_saas/issues/new

Include a job ID, package ID, UTC timestamp, browser version and exact safe error
code when relevant. Never include passwords, API keys, email-verification or
password-reset links, PHI, personal data, real patient data, proprietary
detector output, or source code.

## Response and availability

- Support is normally reviewed within three business days.
- The private beta has no guaranteed uptime, recovery-time or response-time SLA.
- Health is published at `/syn_sig_ra/healthz`; component readiness is
  published at `/syn_sig_ra/readyz`.
- Planned maintenance is announced when practicable.
- Users must retain local copies of packages and evidence they need.

## Billing status

The current private beta is free, collects no payment method and never converts
automatically into a paid subscription. Paid service, if introduced, requires
new pricing, notice and explicit opt-in.
